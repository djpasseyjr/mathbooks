BYU MATHBOOKS

Welcome to BYU Math's textbook checkout program. Keep reading for a description of the application or explore the pages to figure it out yourself.

Checkout and Returns

To checkout books, click on the checkout link in the upper right header menu. Fill out the requested information and press submit. It is extremely important to enter the correct ID. The program searches through all students and professors by ID number and each ID number can only be associated with one name. Entering the wrong ID could make it very difficult to find the right entry in the database.

If you accidently enter an invalid ID number or name, you will need to remove the entry and reenter it correctly. First, go to Tools and click on Table of Books. Use the table to find the entry with the miskeyed ID number. Copy this ID number and use the "Search and Remove Books" tool to remove the entry from the database. You will probably want to write down the title of the book, class and email adress before you delete the entry. Finally, use the information you wrote down to check out the book again with the correct name and ID.

Tools

The tools menu has several useful features. The first link, Table of Books, will display every entry in the database. The second link, Search and Remove books allows the user to search and remove books The third link, Email Students/Professors allows the user to automatically generate emails to an individual or group.

Email

To generate emails to students select the "Email Students/Professors" option in the Tools menu. You will be guided through the appropriate steps for sending out emails to groups of people. When sending large amounts of emails, the confirmation page may take a long time to load. Don't be alarmed. It takes a while to generate each email, connect with the server and send the message. Be patient and it should load eventually.

General

This program was designed to run off a Mac server but could be adapted to Windows. It was written for deparment use with the head secretary in mind, but if ran on a server, the office secretaries could use it as well. Direct any question or issues to djpasseyjr@gmail.com.
